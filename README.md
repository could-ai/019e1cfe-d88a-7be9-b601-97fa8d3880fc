# Vietnameasy

Speak Vietnamese from scratch – fun, fast, and free.

Vietnameasy is a cross-platform language learning application designed to take you from absolute beginner to college-level conversational fluency in Vietnamese. The core experience is built around speaking and listening, removing the barrier of complex reading and writing early on.

## Features

*   **Interactive Speaking Mode:** Listen to native-sounding pronunciation, speak into your microphone, and get instant feedback.
*   **Passive Listening Mode:** A dedicated sleep/background mode where lessons play continuously to help you absorb the language.
*   **Progressive Curriculum:** Structured levels from "Preschool" (tones, alphabet, first words) to Advanced/College.
*   **Multiple Profiles & Guest Mode:** Create local profiles for your family or use guest mode without saving data.
*   **Offline Support:** Designed to work entirely offline once the Vietnamese TTS voice is available.
*   **No Ads, Free Forever:** A pure educational tool with no clutter.

## Tech Stack

*   Framework: Flutter (iOS, Android, Web, Desktop)
*   State Management: Provider
*   Local Storage: Shared Preferences
*   Speech Services: `speech_to_text`, `flutter_tts`

## Getting Started

1.  Ensure you have Flutter installed.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to launch the app.
4.  If running on iOS or Android, grant microphone permissions when prompted for the speaking practice to work.
